Atributos de los inputs

Estos atributos se pueden modificar desde el objeto como una propiedad de las etiquetas.
No hace falta hacer uso de set, get o removeAttribute.

    className: muestra lo que contiene el atributo class
    value: muestra el valor del atributo
    type: muestra el tipo de atributo
    accept: usualmente se usa con type file
    form: cuando hay un formulario existente, se puede ligar un dato de ingreso para que sea guardado en un servidor
    minlengh: es la minima cantidad de caracteres que debe tener un input
    placeholder: coloca un texto en el input "text" para dar alguna indicacion
    required: hace una validación de datos en un input text. No envia campos vacíos por ejemplo.
