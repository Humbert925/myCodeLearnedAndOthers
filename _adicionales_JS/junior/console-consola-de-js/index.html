Consola de Javascript

La consola de javascript sirve para detectar errores.
Existen al menos 3 situaciones muy comunes donde es necesario detectar los errores.
1. hay errores que se muestran pero no se solucionan tan facil como a veces lo señala la documentación.
en ese caso puedes ingresar a la consola, detectar el error y buscar las posibles soluciones desde dentro de la consola.


2. Se detectó el error pero la consola está muy cargada y hay cosas que no deseamos que vea el usuario.
Otra solucion es limpiar la consola y dejarla preparada para que se pueda usar nuevamente.

3. Excelente, pero hay una variable que está cambiando constantemente, por botones y funciones. Me es difícil identificar el valor real.
Sólo hay que ingresar a la consola y pedir que se muestre el valor de esa variable por consola y mostrará el valor de ese momento para esa variable


Dentro de los navegadores puedes abrir la consola con click derecho y haciendo click en inspección o apretando F12.

Funciones de registro

.assert Aparece un mensaje de error en la consola si la affirmación es falsa. Si la afirmación es verdadera, no aparecerá nada.
.clear Limpia la consola.
.error Muestra un mensaje de error en la consola web.
.info Emite un mensaje informativo a la consola web. En firefox y chrome, se muestra un pequño icono "i" junto a estos elementos en el registro de la consola web
.log Muestra un mensaje en la consola web (o del interprete de javascript).
.table Esta funcion toma un argumento obligatorio: data, que debe ser un array o un objeto, y un parámetro adicional: columns y nos muestra una tabla en consola.
.warn Imprime un mensaje de advertencia en la consola web.
.dir Despliega una lista interactiva de las propiedades del objeto javascript especificado

Funciones de conteo

.count Registra el numero de veces que se llama count(). Esta funcion tomo como argumento optional una etiqueta.
.countReset Resetea el contador 'console.count()'

Funciones de agrupación

.group sirven para crear un grupo, esta funciona como "etiqueta de apertura" y console.groupEnd como la de cierre
.grupEnd
groupCollapsed
