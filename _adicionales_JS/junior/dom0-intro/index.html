Document Object Model DOM (introducción)

Introducción

Es como una interfaz que contiene todos los objetos estándard que nos permite representar un documento HTML, XML o XTML.
Que a su vez tiene otra interfaz, que nos permite decir qué cosa combina con cada cosa y otra que nos permite combinarla
con un lenguaje como javascript. Es el documento que contiene todos los elementos HTML, estilos, atributos, TODO.

Nodo: Un nodo en el DOM es cualquier etiqueta del cuerpo en el mismo body o incluso las etiquetas de una lista.

    Document: el nodo document es el nodo raíz, a partir del cual se derivan el resto de nodos.
    Element: nodos definidos por etiquetas html
    Text: el texto dentro de un nodo element se considera un nuevo nodo hijo de tipo text.
    Attribute: los atributos de las etiquetas definen nodos, (js no los ve como nodos :v LOL) sino como información asociada al tio de nodo element
    Comentarios y otros: los comentarios y otros elementos como las declaraciones doctype en cabecera de los documentos HTML generan nodos.
